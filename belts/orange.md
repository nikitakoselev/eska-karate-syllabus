# Orange Belt

**Color:** orange  
**Kyu:** 8  

## Requirements

### Kihon

- **Choku Zuki**  
  Straight punch - from natural stance  
  *References:* Heian Shodan

- **Gyaku Zuki**  
  Reverse punch - stationary from front stance  
  *References:* Heian Shodan

- **Oi Zuki**  
  Lunge punch - stepping in front stance  
  *References:* Heian Shodan

- **Age Uke**  
  Rising block - stepping in front stance  
  *References:* Heian Shodan

- **Ude Uke**  
  Outside block - stepping in front stance  
  *References:* Heian Shodan

- **Uchi Uke**  
  Inside block - stepping in front stance  
  *References:* Heian Shodan

- **Shuto Uke**  
  Knife hand block - stepping in front stance  
  *References:* Heian Shodan

- **Mae Geri**  
  Front kick - stepping in front stance  
  *References:* Heian Shodan

- **Yoko Geri Keage**  
  Side snap kick - stepping in side stance  
  *References:* Heian Shodan

- **Yoko Geri Kekome**  
  Side thrust kick - stepping in side stance  
  *References:* Heian Shodan

### Kumite

- **Kihon Ippon Kumite - SET 1**  
  Basic one step sparring - both sides  
  *References:* Heian Shodan
